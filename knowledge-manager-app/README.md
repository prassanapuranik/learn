# Knowledge Manager App

Simple Vue + Vuex + Axios + Vue Router + Quasar web app for event knowledge management.

## Features
- Two login roles: admin and viewer
- Admin CRUD for events
- Simulated reminder/email sending
- Dashboard metrics
- Monthly calendar view
- Reports by topics/presenters and credit points
- Seeded mock data loaded from `public/db/*.json`

## Run
```bash
npm install
npm run dev
```

Login credentials:
- `admin` / `admin123`
- `viewer` / `viewer123`
